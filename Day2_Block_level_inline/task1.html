Block level tags

    P tag defines a paragraph

    Ul tag defines unordered list

    div tag defines a section or division

    li tag defines a list item

    ol defines ordered list

    hr tag defines a horizontal rule

    address tag defines the contact information of the document

    h1 -h6 tags defines different levels ofheading

    table defines to create a table

    form tag defines html forms

Inline tags

1.Strong tag is used for bold text 2. em tag is used for italic the text 3.span tag is used to mark up a part of a text 4.a tag defines a hyper link 5. img tag is used to insert an image 6.sub tag is used to subscipt text 7. sup tag is used to superscript text 8.time tag defines time 9. u tag is used to underline text 10. q tag is used to add quotations
